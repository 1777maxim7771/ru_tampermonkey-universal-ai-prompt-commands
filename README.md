# Tampermonkey Universal AI Prompt Commands RU

**Tampermonkey Universal AI Prompt Commands RU** — русская версия пользовательского скрипта для расширения **Tampermonkey**. Скрипт помогает быстрее работать с чатами искусственного интеллекта: ChatGPT, Gemini, Claude, Copilot и другими сайтами с полем ввода.

Главная задача скрипта — заменять короткие команды `R1–R10` на заранее подготовленные длинные промпты на русском языке. Благодаря этому не нужно каждый раз вручную писать один и тот же большой запрос.

---

## Для чего нужен скрипт

Скрипт нужен для быстрого ввода готовых промптов в AI-чатах.

Он помогает быстро выполнять типичные задачи:

- перевод текста на русский язык;
- перевод текста на немецкий язык простым уровнем A2-B1;
- краткое описание письма в одну строку;
- суммаризация письма, документа или сообщения;
- объяснение официального текста простыми словами;
- извлечение дат, сумм, имён, организаций, сроков и требований;
- составление списка действий по письму или документу;
- подготовка официального ответа или письма.

---

## Как работает скрипт

Пользователь вводит точную команду в поле ввода AI-чата.

Например:

```text
R1
```

Скрипт автоматически заменяет эту команду на полный русский промпт для точного перевода текста на русский язык.

Ещё примеры:

```text
R3
```

заменяется на промпт для краткого тематического резюме письма в одну строку.

```text
R8
```

заменяется на промпт для извлечения важных фактов из текста.

```text
R9
```

заменяется на промпт для составления списка действий.

---

## Команды русской версии

- `R1` — точный перевод текста на русский язык.
- `R2` — суммаризация текста на русском языке.
- `R3` — краткое тематическое резюме письма в одну строку.
- `R4` — перевод текста на немецкий язык уровнем A2-B1.
- `R5` — исправление русского текста с сохранением смысла.
- `R6` — короткий официальный ответ на немецком языке.
- `R7` — объяснение текста простыми словами на русском языке.
- `R8` — извлечение важных фактов из текста.
- `R9` — список необходимых действий по тексту.
- `R10` — официальное письмо на немецком языке на основе исходного текста.

В этой русской версии комментарии, описание, уведомления и основные промпты оформлены на русском языке.

## Что нужно перед установкой

Перед установкой скрипта в браузере должно быть установлено расширение **Tampermonkey**.

Tampermonkey — это расширение, которое позволяет устанавливать и запускать пользовательские скрипты `.user.js`.

---

## Быстрая установка

1. Установите расширение **Tampermonkey** в браузер.
2. Откройте Raw-ссылку на файл скрипта:

```text
https://raw.githubusercontent.com/1777maxim7771/ru_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js
```

3. Tampermonkey должен автоматически открыть окно установки.
4. Нажмите **Install / Установить**.
5. Откройте AI-чат и введите:

```text
R1
```

Если команда заменилась на длинный промпт, скрипт установлен правильно.

---

## Установка через GitHub

1. Откройте репозиторий.
2. Откройте файл:

```text
tampermonkey-universal-ai-prompt-commands.user.js
```

3. Нажмите **Raw**.
4. Tampermonkey должен распознать файл `.user.js` и предложить установку.
5. Нажмите **Install / Установить**.

---

## Если Raw не сработал

Откройте Tampermonkey вручную:

```text
Tampermonkey → Dashboard → Utilities → Import from URL
```

Вставьте Raw-ссылку:

```text
https://raw.githubusercontent.com/1777maxim7771/ru_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js
```

---

## Ручная установка

1. Откройте Tampermonkey.
2. Нажмите **Create a new script / Создать новый скрипт**.
3. Удалите стандартный шаблон.
4. Скопируйте весь код из файла `tampermonkey-universal-ai-prompt-commands.user.js`.
5. Вставьте код в Tampermonkey.
6. Сохраните через **Ctrl + S**.

---

## Важное пояснение

Скрипт устанавливается не в GitHub и не в конкретный сайт. Он устанавливается в расширение **Tampermonkey**.

GitHub используется только как место хранения файла скрипта.

В скрипте указано:

```javascript
// @match        *://*/*
```

Это значит, что скрипт может работать на разных сайтах, но заменяет только точные команды `R1–R10`. Обычный текст не изменяется.
