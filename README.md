# Tampermonkey Universal AI Prompt Commands RU

**Tampermonkey Universal AI Prompt Commands RU** — это пользовательский скрипт для расширения **Tampermonkey**, который помогает быстрее работать с чатами искусственного интеллекта.

Главная задача скрипта — автоматически заменять короткие команды вроде `R1`, `R3`, `D1` или `D10` на заранее подготовленные длинные AI-промпты. Благодаря этому не нужно каждый раз вручную писать один и тот же большой запрос в ChatGPT, Gemini, Claude, Copilot или другом AI-чате.

---

## Для чего нужен этот скрипт

Скрипт нужен для быстрого ввода готовых промптов в AI-чатах.

Обычно при работе с искусственным интеллектом часто приходится повторять одни и те же запросы:

- перевести текст на русский язык;
- перевести текст на немецкий язык простым уровнем A2-B1;
- кратко описать письмо;
- сделать резюме документа;
- объяснить официальный текст простыми словами;
- извлечь даты, суммы, имена, организации и требования;
- составить список действий по письму или документу;
- подготовить официальный ответ на немецком языке.

Этот скрипт ускоряет такую работу. Пользователь вводит короткую команду, а скрипт сам вставляет полный готовый промпт.

---

## Как работает скрипт

Скрипт отслеживает поле ввода на сайте. Когда пользователь вводит точную команду, например:

```text
R1
```

скрипт автоматически заменяет её на полный промпт для точного перевода текста на русский язык.

Другой пример:

```text
R3
```

будет заменено на промпт для краткого тематического резюме письма в одну строку.

Ещё пример:

```text
R8
```

будет заменено на промпт для извлечения важных фактов из текста: дат, сроков, сумм, имён, организаций, документов и требований.

Команда:

```text
D4
```

будет заменена на немецкую формулировку промпта для перевода текста на простой немецкий язык уровня A2-B1.

## Примеры использования

### Пример 1 — перевод на русский язык

Пользователь вводит в AI-чате:

```text
R1
```

Скрипт заменяет это на полный промпт:

```text
Сделай полный и точный перевод предоставленного текста на русский язык...
```

После этого можно добавить текст письма, документа или сообщения.

---

### Пример 2 — краткое описание письма

Пользователь вводит:

```text
R3
```

Скрипт вставляет промпт, который просит AI сделать краткое тематическое резюме письма на русском языке строго в одну строку.

Это удобно, если нужно быстро понять:

- от кого письмо;
- по какому вопросу;
- что требуется;
- какие есть даты, сроки, суммы или документы.

---

### Пример 3 — список действий

Пользователь вводит:

```text
R9
```

Скрипт вставляет промпт, который просит AI составить понятный список действий по тексту: что нужно сделать, кому ответить, какие документы подготовить и какие сроки соблюдать.

## Где можно использовать

Скрипт в первую очередь предназначен для работы с AI-чатами, например:

- ChatGPT;
- Google Gemini;
- Claude;
- Microsoft Copilot;
- другими сайтами с полем ввода текста.

Скрипт универсальный. Он может работать на разных сайтах, где есть обычное поле ввода, `textarea`, `input` или редактируемое поле чата.

В текущей версии в шапке скрипта указано:

```javascript
// @match        *://*/*
```

Это означает, что Tampermonkey может запускать скрипт на разных сайтах. При этом скрипт выполняет замену только тогда, когда в поле введена точная команда, например `R1`, `R3`, `D1` или `D10`. Обычный текст скрипт не должен изменять.

## Что должно быть установлено перед установкой скрипта

Перед установкой этого скрипта в браузере должно быть установлено расширение **Tampermonkey**.

Tampermonkey — это расширение для браузера, которое позволяет устанавливать и запускать пользовательские скрипты `.user.js`.

Сначала нужно установить Tampermonkey в браузер:

- Google Chrome;
- Microsoft Edge;
- Firefox;
- другой браузер, поддерживающий Tampermonkey.

После установки Tampermonkey можно устанавливать сам скрипт из этого репозитория.

## Быстрая установка

### Способ 1 — установка через Raw-ссылку

Это самый быстрый способ установки.

1. Установите расширение **Tampermonkey** в браузер.
2. Откройте прямую Raw-ссылку на файл скрипта:

```text
https://raw.githubusercontent.com/1777maxim7771/ru_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js
```

3. Tampermonkey должен автоматически открыть окно установки скрипта.
4. Нажмите **Install / Установить**.
5. После установки откройте нужный AI-чат или другой сайт с полем ввода.
6. Введите команду:

```text
R1
```

7. Если скрипт установлен правильно, команда `R1` автоматически заменится на полный готовый промпт.

---

### Способ 2 — установка через GitHub

1. Откройте этот репозиторий:

```text
https://github.com/1777maxim7771/ru_tampermonkey-universal-ai-prompt-commands
```

2. Откройте файл скрипта:

```text
tampermonkey-universal-ai-prompt-commands.user.js
```

3. Нажмите кнопку **Raw**.
4. Tampermonkey должен распознать файл `.user.js` и открыть окно установки.
5. Нажмите **Install / Установить**.

---

### Способ 3 — импорт по URL в Tampermonkey

Если после нажатия **Raw** окно установки не появилось:

1. Откройте **Tampermonkey**.
2. Перейдите в **Dashboard / Панель управления**.
3. Откройте вкладку **Utilities / Утилиты**.
4. Найдите поле **Import from URL / Импорт из URL**.
5. Вставьте Raw-ссылку:

```text
https://raw.githubusercontent.com/1777maxim7771/ru_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js
```

6. Нажмите импорт или установку.
7. Подтвердите установку скрипта.

---

### Способ 4 — ручная установка

Если автоматическая установка не работает, можно установить скрипт вручную.

1. Откройте **Tampermonkey**.
2. Перейдите в **Dashboard / Панель управления**.
3. Нажмите **Create a new script / Создать новый скрипт**.
4. Удалите стандартный шаблон.
5. Откройте в этом репозитории файл:

```text
tampermonkey-universal-ai-prompt-commands.user.js
```

6. Скопируйте весь код скрипта.
7. Вставьте код в редактор Tampermonkey.
8. Сохраните скрипт через **File → Save** или клавишами **Ctrl + S**.

## Почему Tampermonkey распознаёт этот скрипт

Tampermonkey распознаёт пользовательский скрипт по специальной шапке в начале файла:

```javascript
// ==UserScript==
// @name         Tampermonkey Universal AI Prompt Commands RU
// @match        *://*/*
// ==/UserScript==
```

Также файл имеет расширение:

```text
.user.js
```

Поэтому при открытии Raw-файла Tampermonkey понимает, что это пользовательский скрипт, и предлагает установить его в расширение.

Важно: скрипт устанавливается **не в GitHub** и **не в конкретный сайт**. Он устанавливается именно в расширение **Tampermonkey**. GitHub используется только как место хранения файла скрипта.

## Группы команд

### Русские команды `R1–R10`

- `R1` — точный перевод текста на русский язык.
- `R2` — суммаризация текста на русском языке по смыслу и контексту.
- `R3` — краткое тематическое резюме письма на русском языке строго в одну строку.
- `R4` — перевод текста на немецкий язык простым уровнем A2-B1.
- `R5` — исправление русского текста с сохранением смысла.
- `R6` — короткий официальный ответ на письмо на немецком языке.
- `R7` — объяснение текста простыми словами на русском языке.
- `R8` — извлечение важных фактов: даты, суммы, имена, организации, требования, документы.
- `R9` — список необходимых действий по тексту.
- `R10` — официальное письмо на немецком языке на основе исходного текста.

### Немецкие команды `D1–D10`

- `D1` — немецкая формулировка промпта для перевода текста на русский язык.
- `D2` — немецкая формулировка промпта для суммаризации текста на русском языке.
- `D3` — немецкая формулировка промпта для краткого резюме письма в одну строку.
- `D4` — немецкая формулировка промпта для перевода текста на немецкий уровень A2-B1.
- `D5` — немецкая формулировка промпта для исправления русского текста.
- `D6` — немецкая формулировка промпта для официального ответа на немецком языке.
- `D7` — немецкая формулировка промпта для простого объяснения текста на русском языке.
- `D8` — немецкая формулировка промпта для извлечения важных фактов.
- `D9` — немецкая формулировка промпта для списка необходимых действий.
- `D10` — немецкая формулировка промпта для официального письма на немецком языке.

## Возможные проблемы

### Окно установки не открылось после Raw

Откройте Tampermonkey вручную и используйте способ **Import from URL / Импорт из URL**.

### Команда не заменяется

Проверьте:

- включён ли скрипт в Tampermonkey;
- разрешён ли запуск Tampermonkey на этом сайте;
- обновлена ли страница после установки скрипта;
- введена ли команда точно, например `R1`, без дополнительного текста;
- находится ли курсор именно в поле ввода.

### Скрипт запускается на всех сайтах

Это сделано для универсальности. В шапке скрипта стоит:

```javascript
// @match        *://*/*
```

Если нужно ограничить работу только одним сайтом, можно заменить эту строку, например:

```javascript
// @match        https://gemini.google.com/*
```

или:

```javascript
// @match        https://chatgpt.com/*
```
