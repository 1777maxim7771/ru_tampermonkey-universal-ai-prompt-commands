# Tampermonkey Universal AI Prompt Commands RU

**Tampermonkey Universal AI Prompt Commands RU** — русская локализация Tampermonkey-скрипта для быстрой работы с чатами искусственного интеллекта.

Скрипт заменяет универсальные короткие триггеры `Q1–Q10` на заранее подготовленные AI-промпты. Триггеры не привязаны к языку: пользователь может заменить `Q1`, `Q2`, `Q3` и другие на любые свои слова, команды или фразы.

---

## Для чего нужен скрипт

Скрипт нужен для быстрого ввода готовых промптов в ChatGPT, Gemini, Claude, Copilot и другие AI-чаты. Вместо того чтобы каждый раз писать длинный запрос вручную, достаточно ввести короткий триггер, например `Q1`, и скрипт автоматически вставит полный промпт.

---

## Как работает скрипт

Скрипт отслеживает активное поле ввода на сайте. Если всё содержимое поля точно совпадает с одним из триггеров `Q1–Q10`, оно заменяется на готовый промпт.

Пример:

```text
Q1
```

будет заменено на промпт для точного перевода текста на русский язык.

```text
Q8
```

будет заменено на промпт для извлечения важных фактов из текста.

Обычный текст не изменяется. Например, `Q1 какой-то текст` не будет заменён, потому что это уже не точное совпадение с триггером.

---

## Настройка своих триггеров

Триггеры можно менять прямо в коде скрипта в объекте `COMMANDS`.

Например, можно заменить:

```javascript
'Q1': `...`
```

на:

```javascript
'TRANSLATE': `...`
```

или на любое другое слово. Поэтому `Q1–Q10` — это только стандартные универсальные триггеры по умолчанию.

---

## Где можно использовать

Скрипт предназначен прежде всего для AI-чатов:

- ChatGPT;
- Google Gemini;
- Claude;
- Microsoft Copilot;
- другие сайты с полем ввода текста.

В скрипте указано:

```javascript
// @match        *://*/*
```

Это означает, что Tampermonkey может запускать скрипт на разных сайтах. Замена выполняется только при точном совпадении с триггером.

## Что должно быть установлено перед установкой

Перед установкой скрипта в браузере должно быть установлено расширение **Tampermonkey**.

Скрипт устанавливается не в GitHub и не в конкретный сайт. Он устанавливается именно в расширение **Tampermonkey**. GitHub используется только как место хранения файла `.user.js`.

---

## Быстрая установка через Raw-ссылку

1. Установите расширение **Tampermonkey** в браузер.
2. Откройте Raw-ссылку:

```text
https://raw.githubusercontent.com/1777maxim7771/ru_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js
```

3. Tampermonkey должен открыть окно установки.
4. Нажмите **Install / Установить**.
5. Откройте AI-чат и введите `Q1`.

---

## Установка через GitHub

1. Откройте репозиторий.
2. Откройте файл:

```text
tampermonkey-universal-ai-prompt-commands.user.js
```

3. Нажмите **Raw**.
4. Подтвердите установку в Tampermonkey.

---

## Импорт по URL в Tampermonkey

Если окно установки не появилось:

1. Откройте Tampermonkey.
2. Перейдите в **Dashboard / Панель управления**.
3. Откройте **Utilities / Утилиты**.
4. Найдите **Import from URL / Импорт из URL**.
5. Вставьте Raw-ссылку из этого README.
6. Подтвердите установку.

---

## Ручная установка

1. Откройте Tampermonkey.
2. Нажмите **Create a new script / Создать новый скрипт**.
3. Удалите стандартный шаблон.
4. Скопируйте код из файла `tampermonkey-universal-ai-prompt-commands.user.js`.
5. Вставьте его в редактор Tampermonkey.
6. Сохраните через **Ctrl + S**.

---

## Почему Tampermonkey распознаёт скрипт

Tampermonkey распознаёт скрипт по шапке:

```javascript
// ==UserScript==
// @name         Tampermonkey Universal AI Prompt Commands RU
// @match        *://*/*
// ==/UserScript==
```

и по расширению файла:

```text
.user.js
```

## Команды по умолчанию

- `Q1` — перевод текста на русский язык.
- `Q2` — суммаризация текста.
- `Q3` — краткое резюме письма в одну строку.
- `Q4` — перевод на простой немецкий язык A2-B1.
- `Q5` — исправление русского текста.
- `Q6` — короткий официальный ответ.
- `Q7` — объяснение текста простыми словами.
- `Q8` — извлечение важных фактов.
- `Q9` — список необходимых действий.
- `Q10` — официальное письмо на немецком языке.

---

## Как проверить после установки

Откройте любой AI-чат и введите:

```text
Q1
```

Если скрипт установлен правильно, `Q1` заменится на полный промпт.

---

## Возможные проблемы

Если команда не заменяется, проверьте:

- включён ли скрипт в Tampermonkey;
- обновлена ли страница после установки;
- введён ли триггер точно, например `Q1`, без дополнительного текста;
- разрешён ли Tampermonkey на этом сайте;
- находится ли курсор в редактируемом поле.
